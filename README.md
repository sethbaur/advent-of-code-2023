# Advent of Code

To get started:

```
$ npm install
```

Usage:

```
$ npm run day {day}

example:

$ npm run day 12

{ a: 1343215, b: 1231 }
```

## Directory Structure

```
advent-of-code-2022/
└── src/
    ├── days/
    |   ├── 00/ # template for additional days
    │   └── 01/ # day one solution code
    │       ├── index.ts
    │       ├── part1.ts
    │       └── part2.ts
    ├── app.ts
    ├── days.ts # add additional days' solutions to the exports here
```
